Container
  • .container-sm|md|lg|xl, default is .container-sm
  • .container, fixed width container
    • Extra Small, 100%
    • Small, 540px
    • Medium, 768px
    • Large, 960px
    • Extra Large, 1140px
  • .container-fluid, full width container
    • Grid System
  • is built with flexbox and allows up to 12 column
  • Grid System Rules
    • Rows must be placed within a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding
    • Use rows to create horizontal groups of columns
    • only columns may be immediate children of rows
    • Column widths are in percentage, so they are always fluid and sized relative to their parent element
  • Grid Classes
    • .col-
    • .col-sm-
    • .col-md-
    • .col-lg-
    • .col-xl-
    • <div class=".container" style="background-color:red">
          <div class="row">
            <div class="col-sm-4" style="background-color:blue">.col-sm-4</div>
            <div class="col-sm-8" style="background-color:yellow">.col-sm-8</div>
          </div>
</div>

    • Background Colors
