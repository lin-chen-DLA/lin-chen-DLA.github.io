Container
  • .container-sm|md|lg|xl, default is .container-sm
  • .container, fixed width container
    • Window > 576px, Extra Small, 100%
    • 576px <= Window < 768 px, Small, 540px
    • 768px <= Window < 992px, Medium, 768px
    • 992px <= Window < 1200px, Large, 960px
    • Window > 1200px, Extra Large, 1140px
  • .container-fluid, full width container
    • Grid System
  • Built with flexbox and allows up to 12 column
  • Grid System Rules
    • Rows must be placed within a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding
    • Use rows to create horizontal groups of columns
    • only columns may be immediate children of rows
    • Column widths are in percentage, so they are always fluid and sized relative to their parent element
  • Grid Classes
    • .col-
    • .col-sm-
    • .col-md-
    • .col-lg-
    • .col-xl-
    • <div class="container" style="background-color:red">
          <div class="row">
            <div class="col-sm-4" style="background-color:blue">.col-sm-4</div>
            <div class="col-sm-8" style="background-color:yellow">.col-sm-8</div>
          </div>
</div>
    Colors
  • Text colors
    • .text-muted, .text-primary, .text-success, .text-info, .text-warning, .text-danger, .text-secondary, .text-white, .text-dark, .text-body (default body color/often black) and .text-light
  • Background colors
    • .bg-primary, .bg-success, .bg-info, .bg-warning, .bg-danger, .bg-secondary, .bg-dark and .bg-light
    • Spacing
    • Property
      • m, margin
      • p, padding
    • Place
      • t (top), b (bottom), l (left), r (right), x (left and right), y (top and bottom)
    • Size
      • 0 (0rem), 1 (0.25rem), 2 (0.5rem), 3 (1rem = 16px), 4 (1.5rem), 5 (3rem), auto
    <div class="pt-4">(1.5rem = 24px)</div> # top padding 24px

<div class="my-4">(1.5rem = 24px)</div> # top and bottom margin 24px
