Browsersync
What is Browsersync?
  • Browsersync watches for changes you make to a web page and automatically updates the browser
  • Browsersync also works across multiple devices
    • Installation
  • Install NodeJS 
      sudo apt-get install nodejs npm
  • Install browser-sync 
      npm install -g browser-sync
    • Static Webpages
    browser-sync start --server --directory --files "*" //check all files in the directory
    Localhost
    			
  • browser-sync start --proxy "localhost" --files "*"
