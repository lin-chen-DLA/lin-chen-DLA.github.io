Cloudera Data Science Workbench
Architecture
  • Session, containers running on K8s, temporary
  • Each project has a persistent directory, /home/cdsw
  • Packages installed locate at /home/cdsw/.local and are persistent. In other words, sessions share the installed packages

    • Account
  • Personal account is used to work on projects
  • Collaborators of a personal project are selected from global
  • Team account can add/remove team members
  • Projects in a team can be private, team, or public
  • Collaborators in a team project are selected from team members

    • Code Share
  • Use a git repository to share code between projects

    • Environment Variable
  • Site environment variable, Site Administration > Runtime
  • User environment variable, User Settings > Environment Variables
  • Project environment variable, Project Settings > Advanced
