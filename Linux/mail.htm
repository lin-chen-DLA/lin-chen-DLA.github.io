Mail Server
Setup
  • sudo apt-get update
  • sudo apt-get install apache2, install apache2
  • sudo apt-get install php, install php 7
  • sudo apt-get install postfix, install Postfix mail server as an SMTP server
    • mathcs.ecsu.edu
    • sudo service postfix restart
    • sudo dpkg-reconfigure postfix, configure postfix
  • sudo apt-get install dovecot-imapd dovecot-pop3d, install Dovecot mail delivery agent for POP/IMAP functionality
    • sudo service dovecot restart
  • Use SendGrid as a relay host, Azure blocks port 25
    • SendGrid, eagletiger1, eagle1980****
  • sudo apt-get install squirrelmail, install Squirrel Mail
  • sudo cp /etc/squirrelmail/apache.conf /etc/apache2/sites-available/squirrelmail.conf
  • sudo service apache2 reload
  • sudo a2ensite squirrelmail.conf
  • sudo apt install php libapache2-mod-php, configure apache
  • sudo mkdir -p /var/www/html/myusername
  • usermod -m -d /var/www/html/myusername myusername
  • sudo chown -R myusername:myusername /var/www/html/myusername
  • sudo chmod a+rwxt /var/mail
