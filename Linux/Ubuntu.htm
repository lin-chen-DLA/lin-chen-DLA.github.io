  • LD_LIBRARY_PATH
    • set up share library path, this is global variable, only work for current terminal
    • Share Library
    • add share library to /lib and usr/lib, do not modify /etc/ld.so.conf, need run ldconfig
    • install lib to other directory, add configure list file to /etc/ld.so.conf.d/, then sudo ldconfig
    • run ldconfig once share library has any change
