• dpkg and sudo apt-get command
  • sudo apt-get install software
  • sudo apt-get remove, remove package
  • sudo apt-get remove --purge, remove package and related configuration file
  • sudo apt-get upgrade, upgrade installed package
  • sudo apt-get source, download source code
  • Repositories
  • sudo add-apt-respository ppa:, add repository
  • /etc/apt/sources.list, repository list
  • sudo apt-get update, update repository list
