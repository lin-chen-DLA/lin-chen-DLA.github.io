Factorial of a positive integer n, denoted by n!,
  • n! = n * (n-1) * (n-2) ... * 2 * 1

  • the value of 0! is defined as 1

  • for example: 4! = 4 * 3 * 2 *1 = 24

public class Fun2
{
	/** Calculate factorial of a positive integer
	 * @param n a positive number
	 * @return int
	 */
	public static int factorial(int n)
	{
		int fact = 1;

		for(int i = 1; i <= n; i++)
			fact *= i;

		return fact;
	}

	// Test function
	public static void main(String args[])
	{
		System.out.println(factorial(10));
	}
}

/**
 * Java recursive code for factorial
 */

public class Fun
{
	/** Calculate factorial of a positive integer
	 * @param n a positive number
	 * @return int
	 */
	public static int factorial(int n)
	{
		if(n == 0)
			return 1;
		return n*factorial(n-1); # recursive call
	}

	// Test function
	public static void main(String args[])
	{
		System.out.println(factorial(10));
	}
}
What is Recursion?

  • the corresponding function is called as recursive function
