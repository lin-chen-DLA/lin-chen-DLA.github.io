Project Description

Please create a personal website

The personal website contains at least four pages, an "img" folder to store needed images, a "css" folder to store style files

Each page should be able to link to other pages

Create whatever content you would like to present on each page

How to Submit

Deposite the website to an online repository. Submit the URL to the Discord

How to use an online repository will be covered in the next few weeks

When to Present

Last class of the semester

Who will Present

Each student presents one product

How to Present

Each student has four minutes to introduce his/her website via Zoom
