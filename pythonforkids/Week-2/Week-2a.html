Rules for variable names

  1. Variables names must start with a letter or an underscore, such as: "my_name", "_underscore"
  2. The remainder of your variable name may consist of letters, numbers and underscores, such as: "password1"
  3. Names are case sensitive, "myVar" and "MyVar" are not same
  4. Avod using python keywords as variable names, such as: "for", "if"

Image Processing

Contrast

# download image from url
import urllib
url = 'https://lin-chen-va.github.io/pythonforkids/Week-2/pout.tif'
image=urllib.URLopener()
image.retrieve(url,'pout.tif')

# open image
from skimage import io
girl_img = io.imread('pout.tif') # read image and save it to a variable

# display image
import matplotlib.pyplot as plt
plt.imshow(girl_img)
plt.show()

# contrast enhancement
from skimage import exposure
# use algorithm in skimage to increase the contrast of the image, save the modified image to a variable
girl_contrasted = exposure.equalize_adapthist(girl_img, clip_limit=0.03)
plt.imshow(girl_contrasted)
plt.show()
