PHP Basics
Basics
  • Tags
    • <?php, opening tags
    • ?>, closing tags, prefer to be omitted if the file is pure PHP code
    • <?php
	echo "php tags<br>";
?>
  • Comments
    • // or #, line comment
    • /* ... */, block comment
  • Case sensitivity
    • The names of user-defined classes and functions, as well as built-in constructs and keywords such as echo, while, class, etc., are case-insensitive
    • Variables, on the other hand, are case-sensitive
  • Variables
    • Variable names start with a dollar sign ($)
    • $$, reference the value of a variable whose name is stored in another variable
      • <?php
	$v = "temp";
	$temp = 100;

	echo $$v;//100
?>
    • &, variable reference, create aliases
      • <?php
	$v1 = & $v2;
	$v1 = 10;
	$v2 = 100;

	echo "$v1 $v2\n";//100 100
?>
  • Constants
    • only scalar values—Boolean, integer, double, and string—can be constants
      • <?php
	define('NAME', 'Lin');
	echo NAME;
?>
  • Data types
    • Scalar types
      • boolean
      • integer
      • float
      • string
    • Compound types
      • array
      • object
      • callable
      • iterable
    • Special types
      • resource
      • NULL
