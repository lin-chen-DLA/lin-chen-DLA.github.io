Java Documentation
Syntax
  • Occur only with /** ... */ comments
  • Standalone doc tages, are at the beginning of a comment line
  • Inline doc tags, appear anywhere within a Javadoc comment
  • Javadoc will process comment documentation for only public and protected members
Tags
  • @see, standalone doc tag, hyperlinked to the other documentation
    • @see package.class#member label
    • @see #method(Type, ...), refer current class member
    • @see class, refer class in the same package
    • @see package.class, refer class in the different package
    • @see <a href = "url">, refer a external link
  • @author, add author information with javadoc -author
  • @version, add version information
  • @param parameterName description, parameterName should be same as the parameter name in function
  • @throws fullyQualifiedClassName description, add exception information
  • @return description, add return information
  • {@link package.class#member label}, inline doc tag, add hyperlink
  • {@docRoot}, produces the relative path to the documentation root directory
javadoc
  • -d dir, figure out output directory
  • -author, display author information in generated doc
  • -version, display version information
  • -header, include header in each page
  • -footer, include footer in each page
  • -top, add information on the top of each page
  • -bottom, add information on the bottom of each page
package Doc;

import java.util.*;

/** <h3>Javadoc Tutorial</h3>
 * <br>
 * <ul>
 * <li>Occur only within doc comments</li>
 * <li>Javadoc will process comment documentation for only public and protected members</li>
 * </ul>
 * @see Buick
 * @see Doc.A.Honda
 * @see <a href = "https://docs.oracle.com/javase/7/docs/api/java/lang/String.html">String</a>
 * @see #setMaker(String)
 * @author <a href = "mailto:lchen@ecsu.edu">Lin Chen</a>
 * @author Yanhua Feng
 * @version 1.0
 * @since 09.27.2017
 */
public class Car
{
	private String maker;

	/** Initialize Car with maker name
	 * @param c maker name
	 */
	public Car(String c)
	{
		maker = c;
	}

	/** Get make name
	 * @return maker name
	 */
	public String getMaker()
	{
		return maker;
	}

	/** Set maker
	 * @param m maker name
	 */
	public void setMaker(String m)
	{
		maker = m;
	}

	/** Convert a Car object to string
	 * @return string
	 */
	public String toString()
	{
		return "Car: "+maker;
	}
}
