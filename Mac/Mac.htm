1. Basic Operations
    • Mac Keyboard Shortcuts
    • command+I, display file/folder information
    • Ctrl+← and Ctrl+→, swipe left or right three fingers, switch desktop
    • Command-delete, delete the selected files or folders
    • Command-o, open a selected file
    • Command-n, open a new windows
    • Command-r, refresh a webpage
    • Command+Shift-3, copy the whole screen and save it on the desktop
    • Command+Shift-4, snip the screen and save it on the desktop
    • Command+C, copy
    • Command+V, paste
    • Command+C, Command+Option+V, cut and paste
    • Command+space, spotlight search
    • Sleep/Logout
      • Control+Eject, dialog box
      • Control+Shift+Power, display sleep
      • Control+Command+Power, restart
      • Shift+Command+Q, log out
      • Control+Option+Command+Power, shut down
