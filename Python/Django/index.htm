Basics
  • django-admin startproject myproject, create a Django project
  • python manage.py startapp app, create an application
    • admin.py, admin interface
    • models.py, application models
    • tests.py, unit test
    • views.py, application views
  • Add u'172.28.0.12' to ALLOWED_HOSTS in setting.py
  • python manage.py migrate, create tables or collections for database
  • python manage.py runserver 172.28.0.12:8080, start server
  • Access 172.28.0.12:8080/admin, access admin interface
  • python manage.py createsuperuser, create admin account
