Atom specification
  • Hierarchical Specifiers
    • #, model
    • ., submodel/chain/alternative
    • :, residue
    • @, atom
    select #0:10-12.A@CA # select CA atoms in model 0, chain A, residue 10-12
select :ARG; ~select :215 # select ARG, but not select ARG215
select :72@OE2.B # select alternative B of atom OE2 in residue 72
select :ARG.A,.B # select ARG residues in chain A and all residues in chain B
  • Wild cards
    • * matches all atoms in a residue or all residues in a mode
    • = matches parts of atom or residue names
    • ? matches single characters within residue or atom names
    select :*.A # select all residues in chain A
select :GL= # select GLN and GLU
select :AR? # select ARG
  • Zones
    • z/zr<, select the residues within a specific distance
    • za<, select the atoms within a specific distance
    select :68.A z<4 # select all residues within 4 angstoms to residue 68 in chain A
  • Categores
    • ResProp
      • hydrophobic, positive, negative, polar, charged, small, tiny, aromatics, aliphatic
    • msms cat (surfcat)
      • solvent, ions, ligand, main
    select hydrophobic # select hydrophobic residues
  • Attributes
    • Atom attributes
      • element, color, defaultRadius, name
    • Residue attributes
      • isHelix, isSheet, phi, psi, type
    • Model attributes
      • display
    • select @/element=C # select all carbon atoms
select @/element=C and color=red # select all carbon atoms labeled as red color
select @/defaultRadius>1 and defaultRadius<1.5 # select atoms which has default radius between 1 and 1.5 angstroms
		
    select :/type=ARG and isHelix # select ARG residues in Helix
		
    select #/display
    Commands
  • open
  • select
    • select, select model/chain/residue/atom
    • ~select, not select
    • +, next specification string
    • select/sel/picked, selected atoms
    • names of saved selected
    select :ARG # select ARG residues
select ~sel # select all unselected residues
select :ARG; select :PHE # replace mode, select ARG, then select PHE
select :ARG | :PHE # append, select ARG and PHE residues
select :ARG & :/isHelix #intersect, select ARG in helix
  • namesle, name a selection 
    select :ARG # select ARG residues
namesel temp # name selected residues as temp
color red temp # color selected residues
  • color
    • Built-in colors
    select :ARG # select ARG residues
ribbon sel # ribbon selection
color Ne,r # color ribbon
