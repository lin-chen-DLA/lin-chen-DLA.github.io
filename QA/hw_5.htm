Infix to Postfix
  • Create a program to convert infix expression to postfix using stack
  • Initial Steps
    • Create a stack for operators
    • Create a string for postfix expression
  • Traverse Infix
    • Operand, append it to the postfix expression
    • Operator, push onto the stack, pop operators from stack to the postfix expression if they have higher or equal precedence compared to the current operator
    • Left parenthesis, push onto the stack
    • Right parenthesis, opo operators from the stack to the postfix expression until a left parenthesis is encountered, the left parenthesis is discarded
  • Pop Remainnig Operators
    • Pop all operators remaining in the stack and append them to the postfix expression
  • Operator Precedence
    • (), 4
    • ^, 3
    • */%, 2
    • +-, 1
