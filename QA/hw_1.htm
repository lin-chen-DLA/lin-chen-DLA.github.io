Homework 1
  • Write a program that displays all of the prime numbers from 1 through 100. The program should have a loop that calls the is_prime function

  • generate.py, call the functions defined in package Prime to print prime numbers from 1 through 100, and save these prime numbers in a file called primes.txt
  • primemodule.py
    • is_prime(num), define a function to check if a num is an prime
    • get_primes(n), define a function to return a list containing the primes from 1 to n
  • io.py
    • write_primes(prime_list, file_name), define a function to write a list of primes to a file
