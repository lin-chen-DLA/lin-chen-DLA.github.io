Introduction
Features
  • Multiple dispatch, polymorphic, extensible, and composable
  • Compiled, not interpreted, just-in-time (JIT) compiled using the LLVM compiler framework
  • Interactive, REPL (read-eval-print loop)
  • Straightforward syntax
  • dynamic typing and static typing
  • foreign function interfaces, can interface directly with C/Fortran, C++, Python, R, Java
  • metaprogramming, generate other Julia programs
