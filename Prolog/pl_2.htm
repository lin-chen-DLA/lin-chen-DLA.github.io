p(a)   		/* #1 */

u(b). 		/* #3 */

  • Prolog derivation tree is a depth first traversal of such a derivation tree

    ?- p(X), !.
X = a. /* only producing the first answer, the rest branches have been eliminated */

?- p(X), !, u(Y).

Y = b ;
