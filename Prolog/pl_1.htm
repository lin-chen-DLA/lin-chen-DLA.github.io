factorial(0,1).
  
factorial(A,B) :-  
           A > 0, 
           C is A-1,
           factorial(C,D),
           B is A*D.
  • A Prolog goal is terminated with a period "."
  • The built-in predicate 'listing' will list the program in memory
  • The 'halt' goal always succeeds and returns the user to the operating system.
