  • Container adapters do not inherited from the underlying container type
  • Underlying structures can be deque and list, default is deque

Queue with List
#include <iostream>
#include <queue>
#include <list>

int main(int argc, char *argv[])
{
	std::list<int> v = {1, 2, 3, 4};
	std::queue<int, std::list<int>> q(v);

	q.push(5);
	q.pop();

	std::cout<<"Front: "<<q.front()<<std::endl;//2
	std::cout<<"Back: "<<q.back()<<std::endl;//5

	return 0;
}
  • front of queue needs front of underlying container object
  • back of queue needs back of underlying container object
  • push of queue needs push_back of the underlying container object
  • pop of queue needs pop_front of the underlying container object
