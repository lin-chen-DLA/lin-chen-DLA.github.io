Media
Media Query Syntax
  • consists of a media type and can contain one or more expressions
  • the query is true if the specified media type matches the type of device and all expressions are true
  • When a media query is true, the corresponding style sheet or style rules are applied
  • Unless you use the not or only operators, the media type is optional and the all type will be implied
    • @media not|only mediatype and (expressions) {
  CSS-Code;
}
    Examples
    <!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
	<style>
		.container {width: 80%; min-height: 100px; margin: auto; background-color: silver;}

		@media screen and (max-width: 800px) {.container {background-color: red;}}
		@media screen and (max-width: 600px) {.container {background-color: blue;}}
	</style>
</head>
<body>
	<div class = "container">
		Hello World!
	</div>
	
</body>
</html>
		
    <!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
	<style>
		* {box-sizing: border-box;}
		.container {width: 80%; padding: 10px 0px 10px 0px; margin: auto; background-color: silver; overflow: auto;}
		.item {float: left; width: 25%; border: 1px solid red;}

		@media screen and (max-width: 800px) {.item {width: 50%;} .container {background-color: yellow;}}
		@media screen and (max-width: 600px) {.item {width: 100%;} .container {background-color: dodgerblue;}}
	</style>
</head>
<body>
	<div class = "container">
		<div class = "item">Home</div>
		<div class = "item">Research</div>
		<div class = "item">Teaching</div>
		<div class = "item">Service</div>
	</div>
	
</body>
</html>
