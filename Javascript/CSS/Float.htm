    Clear
  • clear property specifies what elements can float beside the cleared element and on which side
    • none - Allows floating elements on both sides. This is default
    • left - No floating elements allowed on the left side
    • right- No floating elements allowed on the right side
    • both - No floating elements allowed on either the left or the right side
    • inherit - The element inherits the clear value of its parent
  • use the clear property is after you have used a float property on an element
    • <!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
</head>
<body>
	<div style = "width: 60%; height: 300px; border: 1px solid red;">
		<div style = "width: 100px; height: 100px; border: 1px solid blue; float: left;">
			Block 1
		</div>
		<div style = "border: 1px solid green;">
			Block 2
		</div>
	</div>
	<br>
	<div style = "width: 60%; height: 300px; border: 1px solid red;">
		<div style = "width: 100px; height: 100px; border: 1px solid blue; float: left;">
			Block 1
		</div>
		<div style = "width: 100px; height: 100px; border: 1px solid green; clear: both;">
			Block 2
		</div>
	</div>
	
</body>
</html>
    Clearfix Hack
  • If an element is taller than the element containing it, and it is floated, it will "overflow" outside of its container:
    • .clearfix::after {
  content: "";
  clear: both;
  display: table;
}
    Box Sizing
  • box-sizing property allows us to include the padding and border in an element's total width and height
  • By default, the width and height of an element is calculated like this
    • width + padding + border = actual width of an element
    • height + padding + border = actual height of an element
    • Example
    <!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
	<style>
	* {
  		box-sizing: border-box;
	}
	header {width: 90%; height: 100px; background-color: silver; margin: auto; text-align: center; padding: 20px;}
	section {width: 90%; border: 1px solid red; margin: auto; overflow: auto; display: flex; flex-wrap: nowrap;}
	section::after {  content: ""; clear: both; display: table;}
	nav {width: 30%; float: left; border: 1px solid blue; background-color: green;}
	nav ul {list-style-type: none;}
	ul a:link {text-decoration: none;}
	nav ul li {width: 80%; height: 60px; background-color: silver; margin: 5px auto; text-align: center; padding: 20px; border: 1px solid red;}
	article {width: 70%; float: left; border: 1px solid blue; min-height: 300px; padding: 20px; background-color: white;}
	footer {width: 90%; height: 100px; background-color: silver; margin: auto; text-align: center; padding: 20px;}
</style>
</head>
