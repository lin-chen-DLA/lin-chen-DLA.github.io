Grid
  • A grid layout consists of a parent element, with one or more child elements
  • display property is set to grid or inline-grid
  • All direct children of the grid container automatically become grid items
    • Grid Area
    <!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
	<style>
		.item1 {grid-area: header; border: 1px solid blue;}
		.item2 {grid-area: nav; border: 1px solid blue;}
		.item3 {grid-area: article; border: 1px solid blue;}
		.item4 {grid-area: footer; border: 1px solid blue;}

		.grid-container {
			display: grid; 
			grid-template-areas: 
			'header'
			'nav'
			'article'
			'footer';
			grid-gap: 10px;
			background-color: silver;
		}

		@media (min-width: 600px) {
			.grid-container {
			display: grid; 
			grid-template-areas: 
			'header header header header'
			'nav article article article'
			'footer footer footer footer';
			grid-gap: 10px;
			background-color: silver;
		}}
	</style>
</head>
<body>
	<div class = "grid-container">
		<div class = "item1">Header</div>
		<div class = "item2">Nav</div>
		<div class = "item3">Article</div>
		<div class = "item4">Footer</div>
	</div>
</body>
</html>
