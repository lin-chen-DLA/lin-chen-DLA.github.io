Architecture
  • Run a Streamlit script, executes the script from top to bottom, components like text, images, charts, and widgets are displayed in the order they appear in the script
  • Streamlit reruns the entire Python script from top to bottom
    • Whenever modify app's source code
    • Whenever a user interacts with widgets in the app
  • Streamlit use @st.cache_data, @st.cache_resource to avoid redundant computations and resource loading across sessions
  • Use st.session_state, and widget states to maintain state across reruns within the same session
    • import streamlit as st

def main():
    print('Start ...')

    if st.checkbox("Display", key='checkbox'):
        st.markdown('Content', unsafe_allow_html=True)
    else:
        st.markdown(f":smile:")

    st.button('Click', key='button')

    print('End ...')

if __name__ == '__main__':
    main()
