Ollama
Interactive Chat
# load model to .llama/models and start a server
ollama run phi3

# /?, help
Commands
# ollama pull [model], pull a model from Ollma library to .llama/models
ollama pull phi3

# ollama run [model], run and chat with a modle
ollama run phi3

# ollama rm [model], remove a model from local server
ollama rm phi3

# ollama list, list models on local server
ollama list

# ollama cp [model_1] [model_2], copy a model
ollama cp phi3 my_model

# ollama show [model], show model information
ollama show phi3
Ollama Server
# display the process of Ollama server
ps aux | grep ollama

# displary the port of Ollama server
lsof -i -P -n | grep ollama # by default, 11434
API
Python API
from ollama import Client
client = Client(host='http://localhost:11434')

# Generate a response (single turn interaction)
client.generate(model='my_model', prompt='What is the capital of France?') # The capital of France is Paris
